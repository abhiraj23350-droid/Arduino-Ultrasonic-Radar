# 📡 Arduino Ultrasonic Radar

A simple **Ultrasonic Radar System** built using Arduino, HC-SR04 sensor, and a servo motor. This project scans the surroundings and displays detected objects on a radar-style interface using Processing.

---

## 🚀 Features
- Real-time object detection  
- 180° scanning using servo motor  
- Radar visualization on computer  
- Low-cost and beginner-friendly  

---

## 🧰 Hardware Requirements
- Arduino Uno  
- HC-SR04 Ultrasonic Sensor  
- SG90 Servo Motor  
- Breadboard  
- Jumper Wires  

---

## 💻 Software Requirements
- Arduino IDE  
- Processing IDE  

---

## ⚙️ Working Principle
The system uses **ultrasonic waves** to detect objects.

1. The sensor emits high-frequency sound waves  
2. Waves reflect back after hitting an object  
3. Arduino calculates distance using echo time  
4. Servo rotates the sensor for scanning  
5. Data is sent to Processing for radar display  

**Formula:**
Distance = (Time × Speed of Sound) / 2

- Easy to build  
- Cost-effective

---

## ⚠️ Limitations
- Short range (2–400 cm)  
- Accuracy depends on surface type  
- Not suitable for long-distance detection
